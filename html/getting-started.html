Getting Started

Before we really get started with programming, we need to first understand what is programming and what it allows you to do. A computer is a powerful tool that is capable of doing various things. But in order for it to work as we expected, it must follow some set of instructions. If we want to build some software that can be used by someone else, we must provide those instructions ourselves. Essentially, programming is the process of writing these instructions for computers, and they eventually combine into an application that can be used by someone else. Because computers only read binary codes, we must rely on programming languages to communicate with a computer. Programming languages consist of bundles of keywords and syntax that are easier to understand for humans, codes are written in a programming language would eventually be converted into binary codes that computers can read. So the best way to get started with programming is to get familiar with one of the programming languages. Below are few good programming languages to start with.
