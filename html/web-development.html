Web Development

Web development is the process of building applications or services that runs through a web browser. This page will be committed to introducing some technologies like HTML/CSS and other web frameworks used in web development. A web framework is a framework that provides some functionalities that are commonly used, they are meant to automate and speed up the development process. A web application can usually be seen as two parts, the frontend and the backend. The frontend is the part that is visible to the user, like the design of a button or how easy it is to navigate the web page. The backend is the part that manages the data to be presented to the frontend. This website introduces frameworks for both frontend and backend. Click here for a more detailed overview.

HTML (HyperText Markup Language) is a language specifically designed to build web pages, it sets up the structures of the web page. It is usually used with CSS (Cascading Style Sheets) which adds stylings based on the structure provided by HTML. People often describe HTML and CSS as the skeleton and the skin of a website.
